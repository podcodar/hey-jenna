# Hey Jenna Server

## Description

Hey Jenna is an App created by Video Makers to Video Makers
This is a [Nest](https://github.com/nestjs/nest) project.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
